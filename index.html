About Me

Hi there! I'm a senior Computer Science student at the University of Texas at San Antonio, working towards graduating with concentrations in Data Science and Software Engineering. I'm passionate about exploring the intersection of the two subjects that I love: technology and art. Using what I've learned, I aim to create accessible, user-friendly interfaces and data visualizations that showcase the beauty and potential of both fields.

Outside of tech, you can find me working on digital illustrations, 3D modeling, playing video games, exploring fashion, and practicing / performing traditional Indonesian dance.

Projects

Rowdy Chat

A ChatGPT-inspired web-app that allows you to talk to UTSA's Rowdy mascot. Developed with Python using OpenAI and Gradio libraries.

Debug Coffee

An ecommerce website for tech enthusiasts and coffee lovers. Built on WordPress with some custom CSS, deployed via AWS with phpMyAdmin configuration.

CNN & LSTM Models

A set of deep learning models for image and text classification. Made in Python utilizing the TensorFlow Keras API and CIFAR-100 / Reuters datasets.

Chess AI

A program that simulates a game of chess between two "players": Random AI and MinMax AI. Created in Google Colab using mostly pure Python.

Project-V

A video game about an infected player wreaking havoc on their village. Developed in Unity using C# and original Blender + Adobe assets.

Base Conversion Game

An application to test knowledge on base conversion problems at various difficulty levels. Developed using Java with a JavaFX GUI.

Name Generator

A simple website that uses hashing to generate a unique name for the given input. Made with HTML / CSS and Javascript.

Search Algorithms

A Python project to develop informed & uninformed search algorithms for mapping Texas cities. Results are visualized by overlaying graph on top of the TX map.

Interpolation Research

A compilation of research findings in the topic of frame interpolation using AI technologies such as DAIN and RIFE, showcased on a WordPress webpage.
